1. Database

1.1. Installation

Go to PostgreSQL website to obtain a distribution of PostgreSQL server. The program has been intensively tested with PostgreSQL 9.1, however the later version should work as well. If you are running Linux/Unix/BSD, we suggest you to install the PostgreSQL server from the official repository. See the PostgreSQL documentation, if you need help with installing or configuring the server.

The following is a typical setup of a user and a database: 

postgres=# CREATE USER myuser PASSWORD 'mypassword';

postgres=# CREATE DATABASE mydatabase OWNER myuser;

postgres=# \c mydatabase

postgres=# ALTER SCHEMA public OWNER TO myuser;
Note that you also need to modify pg_hba.conf to accept TCP/IP connections from the hosts, where the business tier is going to be run. For example, if the business tier is going to be located on the same host as the database server, you need to have the following configuration file: 
host     mydatabase     myuser     127.0.0.1/32    password
