3. Launching

This page explains how to launch Key2Gym after you have deployed it. See Deployment page for details on deployment.

3.1. Running the JAR file

This application is written in Java, and therefore requires a JRE to be installed in order to run it. Once you have setup a JRE, you can run the application with the following command: 
java -jar key2gym.jar
See the following section for details on available command-line arguments.

3.2. Command-line arguments

The following is the table of available command-line arguments:
Argument Description Example
connection Specifies which connection to use. The value of this option is the connection's code name. This option will prevent connections chooser dialog from appearing. java -jar key2gym.jar --connect=mycodename
migrate Causes the database's schema to migrate to the specified version. If latest is specified, the application will migrate the schema to the latest version available. java -jar key2gym.jar --migrate=latest
