Key2Gym is a free software for running and maintaining gyms and other membership-based clubs or enterprises. This application has been written with the ideas of simplicity and powerful functionality in mind.
This application is not intended to be deployed by common users. Amateur attempts to deploy this application in production may result in serious data and, therefore, profit loss. This document provides information to be read by IT specialists responsible for deploying this application.
