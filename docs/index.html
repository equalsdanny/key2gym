Key2Gym is a free software for running and maintaining gyms and other membership-based clubs or enterprises. This application has been written with the ideas of simplicity and powerful functionality in mind. This documentation describes the installation, configuring and administration of the Key2Gym application.
The application consists of 3 tiers: the database tier, the business tier, and the client tier. This documentation is therefore separated into 3 parts. Each part is dedicated to installation, configuration and maintaining of one of the tiers.
