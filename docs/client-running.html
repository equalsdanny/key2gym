4. Client

4.3. Running

This page explains how to launch Key2Gym after you have deployed it. See Deployment page for details on deployment.

Startup script

To run the client you use a startup script located in the client's root directory. Here is an example: 
./key2gym-client.sh
See the following section for details on available command-line arguments.

Command-line arguments

The following is the table of available command-line arguments:
Argument Description Example
connection Specifies which connection to use. The value of this option is the connection's name. java -jar key2gym.jar --connection=default
