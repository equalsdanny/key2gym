3. Client

3.1. Installation

This page explains how to install the Key2Gym client.
The client requires Java Runtime Environment (JRE, JVM, Java VM) to be available at the target computer. Go to Java website for specific instructions on how to install Java Runtime Environment on your OS. All examples of running the client in this documentation assume you have the java program on the path.
  1. Copy the client subfolder from the distribution.
  2. Copy library files listed in client/lib/runtime-deps.txt from lib/ to client/lib/
  3. Configure the client as explained on the Client -> Configuration page.
  4. You can now run the program as explained on the Client -> Running page.
