1. Deployment

This page explains how to deploy a distribution of Key2Gym.

1.1. Java

This program is written in Java. Therefore, it requires Java Runtime Environment to be available at the target computer. Go to Java website for instructions on how to install Java Runtime Environment on your target OS. All examples of launching the application in this documentation assume you have the java program on the path.

1.3. Key2Gym

  1. Unzip the distribution archive to the target folder.
  2. Configure the application as explained on the Configuration page.
  3. Migrate the database schema to the latest version. See Launching page for details.
  4. You can now run the program as explained on the Launching page.
