3.2. Configuration

Running the OpenEJB server is easy, however, note that the database server should already be running upon the OpenEJB starup. Go to bin/ and issue the following command:
  • Linux: ./openejb start to start and ./openejb stop to stop.
  • Windows: Start.bat to start and Stop.bat to stop. See readme files for information on support of Windows Services.
